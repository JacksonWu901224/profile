## 人生七課

### 第一課:

**你沒有那麼多的觀眾，不妨大膽一點。**

### 第二課:

**你不誰最好的朋友，誰也不是誰的唯一。**

### 第三課:

**接受自己的平庸和不完美，人生是用來體驗的，不是用來演繹完美的。**

**你永遠是最好的自己，就算所有人不看好你，也不要去否定自己。**

### 第五課:

**學會拒絕，生活中大多數麻煩，多是來源於不懂拒絕，面對別人的請求，我們本就有拒絕的權利。**

### 第六課:

**世界上沒有比健康更好的財富。**

### 第七課:

**愛人先愛己，擇人先擇新。**

## 道德經

**反者道之動，弱者道之用。**
**天下萬物生於有，有生於無。**

**人惡人怕天不怕，人善人欺天不欺;善惡到頭終有報，只爭來早與來遲。**

**欲知前世因，今生受者是;欲知來世果，今生做者是。**

**菩薩畏因，眾生畏果。**

**信因果者，其心常畏，畏則不敢為惡；**
**不信因果者，其心常蕩，蕩則無所忌憚。**
**一人畏而行一善，萬人即增萬善；**
**一人蕩而造一惡，萬人即增萬惡。**
**故曰人人知因果，大治之道也；人人不信因果，大亂之道也。**

## 一句頂一萬句

**未經他人事，莫論他人非，未履他人途，莫笑他人拙，未經他人苦，莫勸他人善，你若經我苦，未必有我善。**

**你若非我，無權置喙，你若是我，未必如我。**

**別問我過得好不好，不好你也幫不了，好也不是你的功勞。**

**別拿你的三觀，來衡量我做的事。**

**自己不醒悟，他人如何渡;自己若醒悟，何需他人渡。**

**我喜歡自己仁至義盡後，心安理得的無情無義。**

**人最大的愚蠢莫過於話多，最高的境界莫過於沈默，最強的能力莫過於忍耐，最好的修養莫過於穩重;最傻的行為是炫耀，最糟糕的毛病是衝動，最深的智慧是裝傻，最貴的品質是靠普，最該戒的惡習是抱怨，最該學的本領是閉嘴。**

-----------------------------------------------------------------------------

**心安即是歸處: 人生不能左右的事情太多了，人人都有苦衷，事事都有無奈。既然萬事難以周全，又何苦為難自己，學會對自己說一句: "盡力而為，量力而行"。站在自己的高度，看該看的風景。別人的萬頃星河固然壯闊，你窗前的一盞燈火也同樣溫暖。**
**心乾淨，活得就舒服，人實在，過得就踏實。做人無須緊盯他人是善是惡，守住自己內心的一盞良善，便是最大的明亮。待人溫柔幾分，遇事糊塗一點，你終會明白，放過他人，恰恰是放過了那個被困住的自己。生活從來不是攀爬一座叫"別人"的高山，而是耕耘一片叫"自己"的庭院，不嫉妒、不攀比、不強求，靜下心來，你會發現，風有風的自由，雲有雲的溫柔，各有各的渡口，各有各的歸舟。在這紛繁世間，認真做個快樂的普通人，已是最了不起的成就。**
**人生就像一本書，翻開是故事，合上是回憶。**

-----------------------------------------------------------------------------

**唐吉訶德: 我的長槍被大雨磨鈍了，我的戰馬也生鏽了，但我的衝鋒是「唐吉訶德」式的衝鋒，名為「生活」的大風車啊，我要和你大戰三百回合，榮耀即是生命，憐憫勝於征服，勇氣直面深淵，公正如同天秤，靈魂璀璨前程，忠誠永不腐朽，謙遜鑄就豐碑，犧牲點燃永恆，拿起我們的武器，抓好坐騎的韁繩，看到前面那個巨人了嗎？衝鋒！理想的破矛刺向生活的銅牆鐵壁，碰撞出星火，星火略過虛無，露出一絲荒誕，那麼你們憑什麼證明，你們匍匐的世界比我更加高貴。**

-----------------------------------------------------------------------------

**當下的力量: 人生的意義是什麼，其實大部分人的一生都是沒有意義的，三代人以后，也就是百年之后，時間會抹平你存在過的所有痕迹，而你活著的這短短几十年的體驗，就是人生的意義，所以，別讓不開心充斥著我們短暫的一生，要用心去品嚐每一口飯菜，盡情去欣賞每一處風景，勇敢的去熱愛生活，因為這個世界我們只來一次，什麼財富，什麼面子，什麼壓力，什麼人情世故，其實都沒有想象的那麼重要，重要的是珍惜你活著的每一天。**

-----------------------------------------------------------------------------

**查理九世: 你所浪費的今天，是昨日死去之人，苦苦奢望的明天，你所厭惡的現在，是未來的你，再也回不去的曾經，珍惜你所擁有的每一天，才會擁有一個，永無遺憾的生命，衡量人生價值的，並不是生命的長度，而是生命的寬度，當你的能力，還撐不起你的夢想時，那就沉下心來歷練，過往的經歷，不過是樂曲的前奏，當下才是真正的開始。**

-----------------------------------------------------------------------------

**自控力: 如果你不花時間，去創造你想要的生活，那麼你將被迫花很多時間，去應付你所不想要的生活。成功的路上，沒有人會叫你起床，也沒有人會為妳買單，你需要自我管理，自我約束，自我突破，一個人的斗志，是最容易被等待和拖延壓垮的，猶豫不決，是你人生最大的敵人，不管在什麼情況下，我們都應該把精力，用來提升自己，其他的就盡人事聽天命，請記住，在全神貫注修煉自我的時刻，全世界都會為妳，讓出通向陽光的道路。**

-----------------------------------------------------------------------------

**自控力: 在這個世界上，能夠堅持學習，持續進步的人真的是少之又少，看似是千軍萬馬在過獨木橋，其實你真正的對手從來就只有你一個，你的懶惰，你的遲疑，你的猶豫和不堅定，這條路上也永遠只有你自己在場，沒有人會來，因此，你必須學會自我管理，自我約束，自我突破。一個人的鬥志最容易被等待和拖延消磨，可以說猶豫不決是你人生最大的敵人，不管在什麼情況下，我們都應該把精力用來提升自己，要和自己站在一起，去解決問題，而不是和問題站在一起來打倒自己。**

-----------------------------------------------------------------------------

**人性 : 我現在不和人爭吵了，因為我開始意識到每個人只能站在自己的角度看問題，所以如果有人跟你說檸檬是甜的，你只需要對他說是的，那你多吃點兒。沒必要和別人爭辯什麽，因為認知不同，所處的角度也不同，做好自己即可，畢竟在烏鴉的世界裡，天鵝也是有罪的。認知相同，無需爭辯，認知不同，何須爭辯。思想不在一個高度，沒必要互相征服，三觀不在一個層次，微笑就好，畢竟少些無謂爭執，多些內心篤定，焦慮自會慢慢消散。到最後你才懂，沉默不是輸，而是看透後的一種智慧;退讓不是怕，而是格局與修養的體現。能懂你的人，不需要解釋;不懂你的人，說再多也徒勞。不爭，不辯，不求誰懂。時間會讓一切現形，也會讓你明白——看透人性，依然溫柔，才是真正的成熟。**

-----------------------------------------------------------------------------

**當下的力量: 我一直覺得後悔，就是在欺負以前的自己，人生這條路本來就沒有絕對的精準無誤，我不願頻頻回頭，也不想審判當初的自己，沒有抱怨，我坦然為當時的認知買單，沒有自責，我接受所有結果的發生，即使從來一次，以我那時的見識和心境，依舊會走上同一條路徑，我們不該站在今日的高度去指責昨日稚嫩的自己，你要相信，生命中沒有徒勞的旅程，每一段經歷與遺憾，都有它存在的意義，這世上從來不存在天生正確的選擇，是我們一路的努力，讓選擇逐漸成為正確，山友峰頂，海有彼岸，人生漫漫，萬物皆會迴轉，也自有回響。**

-----------------------------------------------------------------------------

**向上生長: 難走的路從來就不壅擠，但是孤獨;好走的路人山人海，但是寂寞。噴泉之所以漂亮，是因為他有壓力，瀑布之所以壯觀，是因為它沒有退路，滴水之所以能穿石，是因為它貴在堅持，人生亦如此——只有堅持別人無法堅持的堅持，才能擁有別人無法擁有的擁有。乾坤未定時，只管向上生長，不要著急否定自己。哪怕風狠狠吹、路遙遙無盡、夜漫漫無邊，也要挺直脊梁——你走的每一步，都在靠近光的方向。**

-----------------------------------------------------------------------------

**與自己和解: 不要把自己逼到抑鬱，也不要把自己氣到一身病，你活著的時候，沒幾個人在意你，你死了以後，也沒有幾個人記得你，所有的事，除了生死，都不值一提，所以請你必須記住，除了生病以外，你所感受到的痛苦，都是你的情緒帶給你的，而非真實存在的，天大的事，只要順其自然，也不過如此，再小的事，倘若斤斤計較，也是寢食難安，你越在意什麼，就會被什麼折磨，萬事從心起，看淡一生輕，心平能育三千氣，心靜可通萬事理，你現在要在乎的，無非三件事：你的錢，你的健康，你內心的平靜，這世界最好的放生，就是放過自己，越安靜就越簡單，別讓脾氣擋住了你的福氣，心靜了，福氣就來了，放下的越多，得到的也就越多，可有可無是心態，可去可留是看淡，取捨之間，便是人生。**

-----------------------------------------------------------------------------

**皮囊: 終有一天你會明白，原來人性最深處的底色，是自私與孤獨，明白了這一點，便不會再向外索求，不再依賴任何人，也不再高估自己在他人心中的位置，人生不過是一場旅行，生活和工作都只是途中的風景，接受一切的發生，允許遺憾、挫敗、無常，甚至允許自己被懷疑和算計，才能真正找到內心的自由。很喜歡這樣一句話:"皮囊不一定完美，但是靈魂一定要高貴，溫柔的一半是知識，另一半則是撐得起你風骨的涵養。"在自己的世界裡獨善其身，在別人的世界裡順其自然，時間終將教會我們洗盡鉛華，回歸本真。**

-----------------------------------------------------------------------------

**屏蔽力: 被別人的話影響情緒，是一件很蠢的事，太在意別人的觀點，只會增加痛苦，鏡子髒的時候，你肯定不會認為是自己的臉髒，那麼為什麼別人評價你不好的時候，你就認為自己是真的不好呢，閒言碎語那麼多，你就做自己就可以了，活在別人的眼裡，就會死在別人的嘴裡，屏蔽力，是一個人最頂級的能力，不去窺探別人的生活，不揣測他人的想法，任何消耗你的人和事，多看一眼都是你的不對，要學會不定義任何人，也不被任何人定義。**

-----------------------------------------------------------------------------

**頓感力: 活著活著就明白了，跟任何人較勁，就是在消耗自己，當無法溝通時就放棄溝通。人和人相處原則是，我們不同，但都沒有錯。不必強求別人完全理解自己，不去計較小事，就隨它去。不要和重要的人計較不重要的事，不要和不重要的人計較重要的事。你的人品是你最好的運氣，你的心態是你最好的風水。**

-----------------------------------------------------------------------------

**人間值得: 煩躁的時候不要說話，也不要做任何決定，安靜的待會兒，人的本質就是活著，不要對別人有太多期待，一些難過的情緒，要學會自己消化，不為難自己不盲目較勁，很多事情你盡力就夠了，好好愛自己，這世上沒多少人真心的心疼你，成年人的煩惱和誰說好像都不合適，悶在心裡容易憋出病，唯有自渡，我們終究還是要自己消化掉，把自己勸明白，就甚麼都通了。**

-----------------------------------------------------------------------------

**情緒: 當你壓力大到快要崩潰的時候，不要跟別人講，也不要覺得自己委屈，沒有人會心疼你，無人問津也好，技不如人也罷，你都要試著安靜下來，去做自己該做的事，而不是讓煩躁和焦慮毀掉你本就不多的熱情和定力，心可以碎，手不能停，該幹甚麼幹甚麼，在崩潰中繼續前行，這才是一個成年人的素養。**

-----------------------------------------------------------------------------

**別對每件事都有反應: 不要在乎與你無關的東西，不要操心你根本就左右不了的事情，不要糾結那些根本就不重要的答案，不要煩惱那些根本就不能把你怎麼樣的人，這個世界就是這樣，你忽略什麼，什麼就消失不見，你介意什麼，什麼就層出不窮。將軍趕路，不追小兔，向日葵要是天天在乎蟲子怎麼叫，就別想朝著太陽開了，當忽略噪音的時候，才能聽見真正的同頻回聲，風會吹過樹林，但每片葉子都有自己的搖晃節奏，不必為了迎合誰的頻率，亂了自己的節拍。你生來與眾不同，無須向任何人證明。**
